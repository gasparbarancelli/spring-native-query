# Security Policy

At this time, there are no known vulnerabilities in this project.

## Reporting Vulnerabilities

If you discover a security vulnerability, please do not open a public issue.  
Instead, send me a detailed email at: gasparbarancelli@gmail.com.  
Please include as many details as possible so I can reproduce and address the issue quickly.

I am committed to responding to and handling all security reports with priority.

Thank you for helping keep this project secure!
